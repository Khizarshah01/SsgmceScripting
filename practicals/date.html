About This Practical

This page demonstrates a simple JavaScript implementation to display the current date. The script uses the built-in Date object to get the current date and formats it using toLocaleDateString() with options for a more readable format.

JavaScript Code:

const date = new Date();
const options = { 
  weekday: 'long', 
  year: 'numeric', 
  month: 'long', 
  day: 'numeric' 
};
const formattedDate = date.toLocaleDateString('en-US', options);
document.getElementById('date').textContent = formattedDate;
